A high-performance, visually stunning personal portfolio application built with Flutter. Designed for seamless cross-platform performance with a focus on modern high-end Android devices (Infinix Hot 50 Pro, Samsung S-Series).

## ✨ Key Features

- **💎 Premium Glassmorphism UI**: Deep blur effects and dynamic transparency for a state-of-the-art feel.
- **🌈 Dynamic Theme Engine**: Seamless transition between dark and light modes with custom color palettes.
- **🎬 Scroll-Triggered Animations**: Fluid entry effects and parallax orbs that respond to user interaction.
- **📱 Responsive by Design**: Flawless layout on every screen size, from small mobile devices to large tablets.
- **🚀 Performance Optimized**: Built with efficiency in mind, using hardware acceleration for smooth 60fps animations.
- **🔍 SEO Ready**: Optimized web version with Open Graph meta tags and structured data.

## 🛠️ Tech Stack

- **Framework**: [Flutter](https://flutter.dev) (Latest Stable)
- **State Management**: [Provider](https://pub.dev/packages/provider)
- **Animations**: [flutter_animate](https://pub.dev/packages/flutter_animate)
- **Icons**: [FontAwesome Icons](https://pub.dev/packages/font_awesome_flutter)
- **Typography**: [Google Fonts (Outfit)](https://fonts.google.com/specimen/Outfit)
- **Visibility Tracking**: [Visibility Detector](https://pub.dev/packages/visibility_detector)

## 📦 Getting Started

### Prerequisites
- Flutter SDK (>=3.0.0)
- Android Studio / VS Code
- Git

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/farmanullah1/Portfolio-Flutter.git
   ```

2. **Install dependencies**
   ```bash
   flutter pub get
   ```

3. **Run the app**
   ```bash
   # Run on connected device
   flutter run
   
   # Run for web
   flutter run -d chrome --web-renderer canvaskit
   ```

## 📂 Project Structure

- `lib/home.dart`: Main scaffold and navigation logic.
- `lib/sections.dart`: Individual UI sections (Hero, About, Skills, etc.).
- `lib/theme.dart`: Design system, colors, and theme data.
- `lib/models.dart`: Portfolio content data source.
